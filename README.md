# code-breakers-The-Manchester-Hiest
an interactive cybersecurity escape room web application designed to teach students and beginners basic cybersecurity concepts through immersive gameplay and investigation-based puzzles.  Players work in teams to investigate a fictional cyberattack, solve challenges, uncover access codes, and stop the breach before the timer expires.

Gameplay Overview
Stage 1 — Phishing Email Investigation
Players analyse a suspicious email and identify warning signs commonly used in phishing attacks.
Stage 2 — Security Investigation
Players investigate security evidence and solve logic-based cybersecurity challenges.
Stage 3 — Password Deduction
Players use clue elimination and reasoning to uncover a hidden password.
Stage 4 — Final Security Challenge
Players complete the final investigation to secure the network.
